    1）flex布局使得元素随内容变化大小，使其中包含的元素都变成inline-block
    2) 如果当前元素和父元素都是flex, 那么当前元素设置justify-content无效，它始终作为父元素唯一的子元素，所以是一个整体并尽量紧靠
